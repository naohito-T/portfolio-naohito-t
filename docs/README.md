# Introduction

projectのDocumentは作成手順とopenAPI2つに分けている。
firebaseへのアクセスについてはOpenAPIを使用し作成
project作成手順などはgitbookに記載

## OpenAPI Document start

```$ make```

## gitbook Document start

```$ make book.start```

## gitbook Document build

```$ make book.build```

## docbook about
